# Homotopy Type Theory (HoTT)

**Proposed Course based on HoTTEST Summer School 2022**

Homotopy Type Theory is a relatively new branch of mathematics that combines algebraic topology with dependent type theory. This course covers the foundations of HoTT, including identity types, univalence, and higher inductive types, with a hands-on approach using Agda.

## Course Structure

### Unit 1: Foundations
- **Lecture 1:** Dependent types and dependent functions
- **Problem Session 1:** Formalizing $\Pi$-types and application
- **Agda Lecture 1:** Introduction to Agda syntax and basic types
- **Agda Problem Sessions:** Implementing functions over dependent types

### Unit 2: Inductive and Identity Types
- **Lecture 2:** Inductive types (Naturals, Lists, W-types)
- **Problem Session 2:** Recursion and Induction principles
- **Lecture 3:** Identity types and Path induction
- **Problem Session 3:** Groupoid structure of types
- **Agda Lecture 2:** Basic MLTT Types in Agda
- **Agda Problem Sessions:** Proving theorems about equality

### Unit 3: Universes and Σ-types
- **Lecture 4:** Universes and the hierarchy of types
- **Problem Session 4:** Working with type families
- **Agda Lecture 3:** Identity types, $\Sigma$-types, and universes
- **Agda Problem Session:** Dependent pairs and type telescopes

### Unit 4: Equivalences and Contractibility
- **Lecture 5:** Equivalences and the definition of 'isEquiv'
- **Problem Session 5:** Proving functions are equivalences
- **Agda Lecture 4:** Working with the equivalence library
- **Lecture 6:** Contractible Types and Singletons
- **Problem Session 6:** Contractibility of fibers
- **Agda Lecture 5:** Implementing contractibility proofs

### Unit 5: Fundamental Theorems and Truncation
- **Lecture 7:** The fundamental theorem of identity types
- **Problem Session 7:** Characterizing identity types of various structures
- **Lecture 8:** Propositions, Sets, and Truncation levels ($n$-types)
- **Problem Session 8:** The hierarchy of propositions and sets
- **Agda Lecture 6:** Calculating the fundamental group of the circle ($\pi_1(S^1) \simeq \mathbb{Z}$)
- **Agda Problem Session:** Proving path equality on the circle

### Unit 6: Advanced Concepts
- **Lecture 9:** Function Extensionality (FunExt)
- **Problem Session 9:** Applications of FunExt in functional programming
- **Agda Lecture 7:** Cubical Agda - an introduction to interval types
- **Agda Problem Session:** Path abstraction and application in Cubical
- **Lecture 10:** Propositional truncation and existence
- **Problem Session 10:** Logic with propositional truncation
- **Agda Lecture 8:** Quotients and Higher Inductive Types (HITs) in Cubical Agda
- **Agda Problem Session:** Implementing the Circle, Torus, and Quotients

### Unit 7: Univalence and Beyond
- **Lecture 11:** The Univalence Axiom ($ (A \simeq B) \simeq (A = B) $)
- **Problem Session 11:** Structure identity principle
- **Agda Lecture 9:** Homogeneous composition and cubical univalence
- **Agda Problem Session:** Working with the Glue constructor
- **Lecture 12:** Univalent combinatorics
- **Problem Session 12:** Isomorphisms in finite set theory

### Special Topics
- **Advanced Session:** How to code your own type theory
- **Advanced Session:** Group theory without groups (Synthetic Homotopy Theory)
- **Advanced Session:** Introduction to modalities
- **Advanced Session:** Cartesian cubical type theory
- **Advanced Session:** Semantics of HoTT in Toposes
